# 🚀 Teste Técnico – Desenvolvedor Full Stack

Projeto desenvolvido como **teste técnico para a vaga de Desenvolvedor Full Stack da SendFlow / Unnichat**.

A aplicação demonstra funcionalidades web modernas utilizando **React, TypeScript e Firebase**, com deploy realizado em **Firebase Hosting**.

👉 [https://teste-vaga-fullstack-luiz.web.app](https://teste-vaga-fullstack-luiz.web.app)

### Frontend

- React
- TypeScript
- Vite

### Backend / Cloud

- Firebase
- Firebase Functions
- Firebase Hosting


---

# 🚀 Executando o Projeto Localmente

### Instalar dependências

```bash
npm install
```

### Rodar aplicação em desenvolvimento

```bash
npm run dev
```

### Gerar build da aplicação

```bash
npm run build
```

### Deploy no Firebase

```bash
firebase deploy
```
